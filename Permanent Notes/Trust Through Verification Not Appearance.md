# Trust Through Verification Not Appearance

## The Shift

| Old Paradigm | New Paradigm |
|--------------|--------------|
| "Does this look real?" | "Under what conditions should this be trusted?" |
| Appearance = Identity evidence | Appearance = Untrusted input |
| One-time authentication | Continuous validation |
| Implicit trust | Explicit, contextual trust |

## Verification Over Recognition

Trust decisions should rely on:
- Cryptographic evidence
- Provenance verification
- Capture integrity
- Cross-channel context
- Temporal consistency

NOT on:
- Visual realism
- Biometric similarity
- Perceptual recognition
